# Отчет по 1 ЛР

### Цель

Научиться создавать и обрабатывать изображения, используя базовые функции библиотеки OpenCV.

### Ход работы

В рамках первой лабораторной работы было разработано консольное приложение для генерации одноканального 8bpp изображения с "градиентной заливкой" и "гамма-скорректированной заливкой". Полоски располагаются друг над другом и имеют параметры s - ширина, h - высота. Гамма-коррекция оформлена в виде функции.

### Работа консольного приложения

Приложение также может принимать параметры s, h, gamma из командной строки. Если ключи не заданы, то используются умолчания (s=3, h=30, gamma=2.4). Имя выходного файла можно указать в виде опционального параметра без ключа, иначе результат отображается на экране, закрываясь по нажатию любой клавиши.

Для запуска консольного приложения, нужно найти исполняемый файл в вашей директории и прописать в консоль:

```.\lab01.exe <s> <h> <gamma> <Путь сохранения файла>```
